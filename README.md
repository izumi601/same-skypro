夜中に眠れないとき、  
不安に押しつぶされそうなとき、  
ただ「誰かがいる」ことを感じられる場所が欲しかった。

SNSを見る余地はない。

**何も話さなくていい、ただ一緒にいる**

## 🌈 コンセプト

### これは何か

- **SNSではない**（投稿もいいねもない）
- **チャットではない**（会話しなくていい）
- **見守りでもない**（監視じゃない）

**「同じ空を見ている人がいる」**  
それだけを知れる場所です。

### 核心的な価値

#### 1. 何も求めない優しさ
投稿も、反応も、言葉も要らない。  
ただ「いる」だけでいい。

#### 2. 静かな寄り添い
「頑張れ」も「大丈夫」も言わない。  
ただ、同じ空を見ている人がいる。

#### 3. いつでも、誰にでも
深夜でも、明け方でも。  
誰でも、いつでも、ただいるだけでいい。

---

## ✨ 主な機能

### 🌅 時間帯で変わる空
朝、昼、夕方、夜、深夜...  
リアルタイムで空の色が変化します。

**時間帯:**
- 04:00-06:00：明け方（紫→ピンク→オレンジ）
- 06:00-10:00：朝（青空）
- 10:00-16:00：昼（明るい青）
- 16:00-18:00：夕方（オレンジ→紫）
- 18:00-22:00：夜（濃い青→紫）
- 22:00-04:00：深夜（深い青）

### ⭐ 星と雲のアニメーション
ゆっくりと流れる雲、瞬く星。  
ただ眺めているだけで、少し落ち着きます。

### 🎵 静かな音楽（選択式）
ピアノの音楽が、あなたの時間に寄り添います。  
嫌なら、音なしでも。

**音楽:**
- 甘茶の音楽工房「春江のあこがれ」
- 音量：30%（控えめに）
- いつでもON/OFF可能

### 👥 今、○人が同じ空を見ています
リアルタイムで表示される人数（3〜17人）。  
**「一人じゃない」** を感じられます。

### ℹ️ 情報ボタン
アプリの説明と、緊急連絡先を掲載。

## 🚀 ローカルで起動する方法

### 前提条件
- Webブラウザ（Chrome、Firefox、Safari等）
- （オプション）VS Code + Live Server拡張機能

### 手順

#### 方法1：直接開く
```bash
# ダウンロードしたindex.htmlをダブルクリック
```

#### 方法2：Live Server（推奨）
```bash
# VS Codeでindex.htmlを開く
# 右クリック → "Open with Live Server"
```

#### 方法3：Python簡易サーバー
```bash
# プロジェクトフォルダで
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## 💭 このプロジェクトから学んだこと

### 1. 「引き算」の大切さ

最初は、チャット機能や投稿機能を考えました。  
でも、**本当に必要なのは「何もしなくていい」こと**だと気づきました。

機能を削ぎ落とすことで、  
**本質的な価値だけが残る**。

### 2. 「寄り添う」とは何か

押し付けない。  
選択肢を与える。  
ただ、そこにいる。

これは、カスタマーサクセスの「伴走型支援」と同じだと感じました。

### 3. 技術は「体験」を作るためにある

アニメーション、色の変化、音楽。  
これらは「技術のデモ」ではなく、  
**「一人じゃない」という体験を作るため**にある。

技術は手段であり、  
**人に寄り添うことが目的**。
